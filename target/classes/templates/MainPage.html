Приветствую, данная страница предназначена для информации о взаимодействии с REST сервисом

Все маппинги сервера доступны по адресу на котором Вы сейчас находитесь. Протестировать сервис можно PostMan программой. Данный сервер производит работу по созданию пользователя, отправки письма для подтверждения регистрации. Помимо этого, существует возможность загрузки изображения для пользователя. Принцип работы основан на использовании ключа JWT. Стек используемых технологий : Spring Boot, Postgresql, Docker, JPA, GIT. Сервер базируется на Raspberry Pi 4.
